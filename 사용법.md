## 깃허브에 저장하는 법

파일 수정하고 나서 아래 순서대로 실행

**1. 터미널 열기**

**2. 작업 폴더로 이동**
```
cd ~/Public/hyc
```

**3. 저장 (커밋 + 푸시)**
```
git add .
git commit -m "여기에 뭘 했는지 적기"
git push
```

예시:
```
git commit -m "링크 보관함 태그 추가"
git commit -m "위젯 색상 변경"
```
